# 구인구직 플랫폼 (MSA 버전)

## 서비스 구성
- 메인 https://github.com/minsoooni/minsoo_final_main
- 보드(커뮤니티) https://github.com/minsoooni/minsoo_final_board
- Discovery https://github.com/minsoooni/minsoo_final_discovery
- 게이트웨이 https://github.com/minsoooni/minsoo_final_gateway

## 기술 스택
Spring Boot / MSA / Eureka / API Gateway / MyBatis / Oracle / Jenkins / Kubernetes

## 담당 파트
- 관리자 회원 관리 기능 (풀스택)
- 배너 관리
- 커뮤니티 관리
- 공지사항 관리
- 공지사항 일

## 프로젝트 구조
- final_main : 메인 서비스 (회원, 관리자)
- final_board : 커뮤니티 서비스
- final_gateway : API Gateway
- final_discovery : Eureka Discovery
